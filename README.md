# Conway's Game of Life Simulator

A desktop simulation of Conway’s Game of Life built with C++ and Qt.  
This project features an interactive grid system, simulation controls, adjustable speed settings, and unit testing support.

---

## Preview

![Game Preview](game-preview.png)

---

## Features

- Interactive grid system
- Start and stop simulation controls
- Step-through generation support
- Adjustable simulation speed
- Save and reset functionality
- Conway’s Game of Life rules implementation
- Unit testing with Qt Test framework

---

## Technologies Used

- C++
- Qt
- CMake
- Qt Test

## Building the Project

### Requirements

- Qt Creator
- Qt 6+
- CMake
- C++17 compatible compiler

### Steps

1. Clone the repository

```bash
git clone https://github.com/Rob2700/game-of-life-qt.git
```

2. Open the project in Qt Creator

3. Open the root `CMakeLists.txt` file

4. Configure the Qt kit if prompted

5. Build and run the project

---

## Running Tests

Unit tests are located in:

GameOfLifeTests/

Run the tests through Qt Creator or your configured test environment.
